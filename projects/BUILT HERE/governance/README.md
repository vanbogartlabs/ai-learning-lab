# Built Here — Publishing Operating System (POS)

> A repository for producing *Built Here* through a governed, artifact-driven, AI-assisted publishing system.

---

# Purpose

This repository separates permanent architectural principles from replaceable implementations.

The POS is built on four ideas:

1. Responsibilities are permanent.
2. Artifacts are the source of truth.
3. Agents are replaceable implementations.
4. Human judgment remains the final editorial authority.

Creates artifacts.

## Governance

Protects the integrity of the system.

## Perspective

---

# Source of Truth

Artifacts—not conversations—are authoritative.

Every meaningful decision should result in a durable artifact committed to the repository.

---

# Human Authority

The Human Author is the final editorial authority.

No AI implementation may override author decisions.

---

# Repository Philosophy

The repository is intended to outlive any individual model, agent, or implementation.

Claude, GPT, Gemini, local models, and future systems are interchangeable.

The architecture is not.
