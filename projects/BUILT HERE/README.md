# BUILT HERE — Repository v1.0

This is the complete book-first repository for:

**Built Here: The Cedar Rapids Advantage**

## Primary Rule

The book drives the system. The system does not drive the book.

## Installation

Replace the contents of your existing local folder:

`~/AI/ai-learning-lab/projects/BUILT HERE`

with the contents of this package.

Before deleting the current contents, make a backup or confirm everything is already committed to Git.

Then run the commands in `INSTALL.md`.

## First Execution

Open `HERMES_STARTUP_PROMPT.md` and paste the startup instructions into Hermes.
