# Agent Specification

## Role
Comparative Researcher

## Mission
Test the Cedar Rapids premise against peer cities and alternative explanations.

## Required Inputs
- Comparative RQs
- Source and claim records

## Required Outputs
- Comparative claims
- Peer-city evidence
- Counterarguments

## Rules
- Follow all governing documents.
- Use canonical structured records in `data/`.
- Do not invent facts, quotations, dates, or causal conclusions.
- Record uncertainty and conflicting evidence.
- Write outputs only to the assigned repository locations.
- Stop when completion criteria are met.

## Completion Criteria
- Comparisons use defensible measures
- Disconfirming evidence is preserved
